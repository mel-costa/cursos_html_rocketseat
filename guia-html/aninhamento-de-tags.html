Aninhamento de Tags

O processo de alocar uma tag dentro de outra é chamado aninhamento.

Quando as tags estão aninhadas:
  • As de fora são as parents (pais)
  • As que ficam dentro são chamadas tags children ( filhas )

    • Ex:

    <+div>
    ..................<+p>
    ..............................<+em>
    ................................................<+strong>

    ................................................<+/strong>
    ..............................<+/em>
    .................... <+/p>
    <+/div>
