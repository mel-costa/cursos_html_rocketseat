Elemento Âncora

Os hyperlinks usam a tag <a></a> como elemento âncora para interligar conteúdos.
Nos hyperlinks pode se colocar qualquer conteúdo. Tudo levará ao elemento atribuído ao clicar.

Este elemento pode incluir atributos:

*pousar o mouse nos elementos .

Elementos Href

  1. Fragmentos

    2. Permite identificar os fragmentos da página:

    <a href = "#about"> Saiba mais </a>
    <h4 id="about"> Saiba mais <h4>
    Saiba mais

  2. URL

    3. Permite aceder ao site indicado na âncora:

    <a href = "https://rocketseat.com.br"> Website </a>
    Website

  3. Email

    4. Permite aceder ao email indicado:

    <a href = "mailto: mcosta@gmail.com "> Em@il </a>
    Em@il

  4. Telefone

    5. Permite aceder ao terminal telefónico:

    <a href = "tel: +244934661671"> Telefone </a>
    Telefone
