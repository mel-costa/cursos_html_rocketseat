Anatomia de um Documento HTML

Para escrever os códigos, os documentos em HTML possuem uma estrutura base:

< ! DOCTYPE html >
     <html>
        <head>
               <title>Estrutura HTML </title>
        </head>
        <body>
               <p>Texto sobre a anatomia dos documentos em html 5 </p>
        </body>
     </html>

*Pousar o cursor nos elementos para ver a descrição
