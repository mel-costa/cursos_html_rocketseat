> URL

Uniform Resourse Locator

-Sequência de texto que localiza recursos na web

URL Absoluto

-Inclui o protocolo e o nome de domínio
-Apontará sempre para o mesmo local

<a href = "https://rocketseat.com.br"> Rocketseat </a>
Ex.:Rocketseat

URL Relativo

-Relativo a página aberta no momento
-Apontará para lugares diferentes

<a href = "https://google.com"> Google </a>
Ex.:Google

> Caminhos

Caminhos de arquivos

-Localiza recursos no explorador de arquivos

Entrando em directórios

-Permite aceder a recursos localizados em diferentes directórios
<a href = "arquivo"> texto </a>
<a href = "../hyperlinks/hyperlinks.html"> Conteúdo sobre Hyperlinks </a>
Ex.:Conteúdo sobre Hyperlinks

Mesmo directório

-Permite aceder a recursos localizados no mesmo directório
<a href = "pasta/arquivo"> Texto </a>
<a href = "Nova/a.txt"> Arquivo de texto a </a>
Ex.:Arquivo de texto a
