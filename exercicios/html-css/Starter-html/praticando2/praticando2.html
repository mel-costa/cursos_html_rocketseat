Praticando imagens

  1. Coloque a seguinte imagem na sua página: https://rocketseat.com.br/icons/icon-48x48.png
  2. Adicione:
    • Um texto alternativo(alt)
    • Um título (title)
    • Uma largura de 500px (width)
    • Removendo a largura, adiciome uma altura de 50px(heigth)
  3. Experimente fazer modificações
  4. Por fim, adicione um link que redirecione a URL https://rocketseat.com.br/ ao clicar na imagem
